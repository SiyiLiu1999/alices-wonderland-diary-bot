# Alice’s Wonderland: Your Private Diary

## 🌟 Overview

Alice’s Wonderland is an offline, browser-based chatbot designed as a safe, non-judgmental space for users to express their thoughts and emotions. It functions like an imaginary friend or personal diary, allowing users to vent, reflect, and process feelings freely.

This project is built entirely with **HTML, CSS, and JavaScript**, with **no external APIs**, ensuring privacy and simplicity.

---

## 🎯 Features

### 💬 Emotion-Aware Chatbot

* Detects emotional tone using keyword matching
* Responds with empathetic, supportive messages

### 🧠 Reflection-Based Interaction

* Encourages deeper thinking with reflective prompts
* Helps users process thoughts rather than just react

### 🫶 Comfort & Support

* Provides comforting replies, encouragement, and grounding tips
* Designed to feel like an imaginary friend

### 🔁 Session Memory (Temporary)

* Remembers previous messages within the session
* Not stored after refresh (ensures privacy)

### 🌍 Translation Feature

* Supports simple translations (English → Spanish, French, German)

### ✨ UX Enhancements

* Typing animation for natural interaction
* Smooth chat interface

### 🔒 Privacy-Focused

* No data storage
* No external API calls
* Chat is cleared upon refresh

## 🧠 Design Logic

The chatbot uses a **rule-based architecture**:

* **Keyword detection** → identifies emotional states
* **Predefined response sets** → generates empathetic replies
* **Randomization** → creates variation in responses
* **Reflection prompts** → encourages deeper thinking
* **Session memory** → references earlier messages

This approach simulates emotional intelligence without using machine learning or APIs.

---

## 🎨 Design Concept

Inspired by *Alice in Wonderland*, the chatbot represents a surreal, safe inner space where thoughts can "wander freely." The interface is designed to be calming, minimal, and accessible across devices.
